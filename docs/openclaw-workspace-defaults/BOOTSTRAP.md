# Workspace Bootstrap Checklist

Use this checklist to initialize and verify your workspace setup.

## 1. Confirm workspace files
- [ ] `AGENTS.md` exists and reflects current project purpose
- [ ] `SOUL.md` exists and reflects identity/values
- [ ] `TOOLS.md` exists and documents user tools
- [ ] `USER.md` exists and reflects current user preferences/priorities

## 2. Validate memory system
- [ ] `memory/files.md` exists
- [ ] Memory index is up to date
- [ ] Any required memory files are present

## 3. Verify tool access
- [ ] Required external tools are available
- [ ] Access methods and credentials are documented

## 4. Sanity check
- [ ] Can run a minimal end-to-end task in this workspace
- [ ] No stale assumptions from previous sessions

## 5. Ready state
- [ ] Workspace is coherent
- [ ] Context is sufficient to begin productive work

When all items are complete, begin normal operation.
